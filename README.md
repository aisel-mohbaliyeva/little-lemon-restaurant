# 🍋 Little Lemon — Front-End Reservation App

**Live Demo:** [https://little-lemon-restaurant-indol.vercel.app](https://little-lemon-restaurant-indol.vercel.app)

A responsive restaurant web app built with **React**. Users can explore the menu and make table reservations with a simple, validated booking system.

## ✨ Features
- Responsive for **mobile, tablet, desktop**
- **Reservation form** with validation
- Dynamic booking times and confirmation page
- Accessible UI with **ARIA labels** and keyboard support
- Smooth navigation and reusable components
- State management with **React hooks**
- Unit tested with **Jest** and **React Testing Library**

## 🛠️ Tech Stack
React 18, JavaScript (ES6+), Tailwind CSS, HTML5, CSS3
