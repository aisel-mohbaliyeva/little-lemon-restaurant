# 🍋 Little Lemon Restaurant — Front-End Reservation App

A fully responsive restaurant web application built with **React**, created as a front-end capstone project.  
The app allows users to explore the restaurant experience and make table reservations through an accessible, validated booking system.

## 📌 Overview

**Little Lemon** is a modern restaurant web app inspired by a Mediterranean restaurant concept.  
This project focuses on building a polished front-end user experience with responsive layouts, form validation, accessibility best practices, and reusable React components.

The main feature of the app is a **reservation booking flow**, where users can select a date, choose an available time slot, enter reservation details, and receive a booking confirmation.

---

## ✨ Features

- Fully responsive design for **mobile, tablet, and desktop**
- **Table reservation form** with client-side validation
- **Dynamic booking times** based on selected date
- Booking confirmation page with submitted details
- Accessible UI with **ARIA labels**, keyboard support, and semantic HTML
- Sticky responsive navigation with mobile menu
- Reusable component-based architecture
- State management using **React hooks** and `useReducer`
- Unit testing with **Jest** and **React Testing Library**
